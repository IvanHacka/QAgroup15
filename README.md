# QAgroup15
# A Software Bug Tracking and Reporting Tool.
This software system is to record and track bugs. Used GitHub issues and Jira as reference=.

## Features

1. Log software bugs or issues.
2. Track bugs status from discover -> fix.
3. Assign to developers.
4. Prioritise based on estimated complexity and duration.
5. Generate reports.

## Prequisities
- python 3.0 or above
- pip

## Installation
```bash
# Clone the repository
git clone [your-repo-url]

# Create virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

## Usage

python main.py
